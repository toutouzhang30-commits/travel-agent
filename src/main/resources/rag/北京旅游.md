北京（帝都）拥有极其庞大且厚重的城市尺度。它的旅游气质是“宏大叙事”与“皇家威严”。
与江南的婉约和小资不同，北京的美在于红墙黄瓦的故宫、庄严肃穆的天安门以及充满京味儿的胡同。
对于游客来说，这里的玩法核心是“阅读历史”和“感受权力美学”。
由于景点面积巨大，北京游玩非常消耗体力，是一场名副其实的“拉练式旅游”。
避雷建议：北京绝不适合说走就走的闲逛，各大核心景点（故宫、国博、毛主席纪念堂等）必须提前数天甚至一周实名抢票，且安检严格，切勿抱有“到了再买票”的侥幸心理。

接上文，北京的景点不仅面积大，且需要极强的预约意识。其旅游资源可划分为四大硬核板块：
1. 核心中轴线：天安门广场、故宫、景山公园、钟鼓楼。这是北京的心脏，是感受明清皇家文化和现代中国庄严感的绝对核心。
2. 皇家园林区（海淀区）：颐和园、圆明园为主。这些庞大的水系园林代表了中国古典园林艺术的最高成就，极其耗费脚力。
3. 老北京胡同区：南锣鼓巷、什刹海、五道营胡同。这里保留了四合院风貌，是体验老北京胡同文化、喝豆汁儿、听相声的集中地。
4. 长城沿线（郊区）：八达岭长城或慕田峪长城。距离市区较远（单程约1.5-2小时），是体验“不到长城非好汉”的必选项，通常需要占据一整天的时间。

北京属于温带大陆性季风气候，气候干燥，日夜温差大，在游览故宫等大型户外景点时需特别留意：
- 秋季（9-10月）：老舍笔下“北平的秋天”，是一年中最完美的旅游季节。秋高气爽，银杏黄了（如雍和宫、地坛），天空湛蓝，极其适合登高和徒步。
- 春季（3-5月）：气温回暖，可赏玉兰和樱花，但春季常伴有大风和柳絮飘飞，偶尔有沙尘暴，敏感人群需准备口罩和防护用品。
- 夏季（6-8月）：暴晒且高温（常超35℃），故宫等广场区域几乎没有树荫遮挡。极易中暑，必须带足饮用水并做好硬核防晒。
- 冬季（12-2月）：干冷刺骨，室外伴有西北风。但若能赶上下雪，“故宫变紫禁城”是绝美体验。室内都有充足暖气，需采用“洋葱式穿衣法”方便穿脱。

考虑到北京气候干燥及景点面积庞大，行程安排必须秉持“量力而行、早起早睡”的原则：
- 4日经典暴走游：Day1 天安门看升旗 + 故宫深度游 + 景山看日落；Day2 颐和园 + 圆明园（挑战极限体力的一天）；Day3 慕田峪长城一日游；Day4 雍和宫祈福 + 什刹海胡同闲逛。
- 节奏避雷与体力分配：北京旅游是地狱级的体力考验。故宫和颐和园单日步数轻松突破2万步。强烈建议每天只安排 1 个大型核心景点，下午搭配 1 个临近的休闲景点（如逛完故宫去什刹海划船）。
- 交通忠告：北京路网横平竖直但非常宽阔，过马路常需走天桥或地下通道。绝对首选地铁出行，避开早晚高峰打车。如果是带长辈或小孩出行，务必削减30%的行程密度。
